Why Choose CloakLM for Document Privacy?

Enterprise-grade PII redaction and document management for secure AI workflows.

🧠

Multi-Pass PII Redaction

Pass 1: GLiNER + IBM Docling scan locally. Pass 2: Optional deep scan with a local Ollama LLM catches what rule-based models miss. No PII ever leaves your device.

📁

Project Management

Organize documents into projects. Add multiple PDFs, run multiple chats against the same document set, and toggle which docs are in context per conversation.

🔑

BYOK — Bring Your Own Key

Connect your own API keys for Claude, Gemini, OpenAI, or run 100% offline with local models via Ollama. You own your data flow.

🛡️

Review & Control

Review every redaction before sending. Accept or dismiss AI suggestions, manually redact additional text, and see provenance badges showing which pass found each PII.

📄

Deep PDF Analysis

Extracts layouts, tables, and complex formatting from PDFs. Attach already-redacted project docs to any chat — no re-processing needed.

🔒

Secure by Design

Anonymized entities (e.g., [PERSON_1]) are restored on-device. API keys stored locally via encrypted Tauri store. No middleman servers, ever.

How CloakLM Secures Your Documents

A multi-pass pipeline ensures nothing slips through.

1️⃣

Upload & Organize

Drop PDFs into a quick chat or organize them into projects for ongoing work with multiple conversations.

2️⃣

Multi-Pass Redaction

Pass 1 (GLiNER) runs instantly. Enable Pass 2 (Deep Scan) for a local LLM to catch names, addresses, and fragments the first pass missed.

3️⃣

Review & Chat

Review all redactions, accept or dismiss suggestions, then chat securely with your chosen AI model. Responses are de-anonymized on-device.
